# Inspector 🔍

> Scan your fridge. Order what's missing.

AI-powered fridge scanner for Indian households. Photo → detect low-stock items → Instamart order in seconds.

**[Live Demo →](https://ankit-mba22014.github.io/Inspector/)**

To be Built on [Swiggy MCP Builders Club](https://mcp.swiggy.com/builders/)

---

## Screens

| # | Screen | Description |
|---|--------|-------------|
| 1 | Home | Upload / photograph fridge |
| 2 | Photo taken | Preview and confirm |
| 3 | Inspecting | AI scanning animation |
| 4 | Results | Order now / Running low / Stocked |
| 5 | Order review | Cart with Instamart prices |
| 6 | Success | Order placed, live tracking |

---

## Run locally

Just open `index.html` in any browser. No dependencies, no build step.

---

## Deploy on GitHub Pages (2 minutes)

1. Create a new GitHub repo called `inspector`
2. Upload `index.html`
3. Go to Settings → Pages → Source: `main` branch → `/root`
4. Your demo is live at `https://yourusername.github.io/inspector`
